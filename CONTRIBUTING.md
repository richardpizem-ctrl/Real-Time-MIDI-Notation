# Contributing to Real-Time MIDI Notation

Thank you for your interest in contributing to this project!  
We welcome all improvements, ideas, bug reports, and feature suggestions.

## How to contribute

### 1. Reporting issues
If you find a bug or have a feature request, please open an issue in the **Issues** tab.  
Describe the problem clearly and include steps to reproduce it if possible.

### 2. Submitting pull requests
- Fork the repository  
- Create a new branch for your changes  
- Make your edits  
- Submit a pull request with a clear description of what you changed and why

### 3. Coding guidelines
- Keep your code clean and readable  
- Comment complex logic  
- Follow the structure of the existing project
