# Security Policy

## Supported Versions
The following versions of this project are currently supported with security updates:

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability
If you discover a security vulnerability, please report it responsibly.

To report a vulnerability, open an Issue on GitHub or contact me directly.

Please include:
- a clear description of the issue
- steps to reproduce
- potential impact

I will review the report as soon as possible.
