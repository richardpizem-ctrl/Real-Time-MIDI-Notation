# Support

Thank you for your interest in **Real‑Time MIDI Notation**!  
This project is in active development, and community support is essential for improving stability, performance, and cross‑platform compatibility.

Below you will find information on how to get help, report issues, and contribute effectively.

---

## 📌 Where to Get Help

If you need assistance, you can use any of the following channels:

### 🔹 GitHub Issues (recommended)
Use the **Issues** tab to report:
- bugs  
- performance problems  
- UI/renderer issues  
- MIDI device compatibility problems  
- feature requests  
- questions about usage  

The project includes professional Issue Templates to make reporting easier.

### 🔹 Email Support
For private or sensitive questions, you can contact the maintainer directly:

**richardpizem@gmail.com**

---

## 🧪 Testing the Project

Because this is a real‑time MIDI prototype, testing on different systems is extremely valuable.

Please test:
- MIDI input latency  
- multi‑track behavior  
- Yamaha arranger compatibility  
- rendering performance  
- UI responsiveness  
- unexpected behavior  

If you test the project, please consider reporting your results.

---

## 🐞 Reporting Problems

When reporting an issue, include:
- OS version  
- Python version  
- MIDI device used  
- steps to reproduce  
- screenshots or logs (if available)

Clear reports help us fix problems faster.

---
